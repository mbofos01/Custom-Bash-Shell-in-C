int isBuilt ( char *  cmd)

Check and execute a built in command.

This function basicaly runs the built in commands of our shell. At this point we only have around 10 commands. exit,logout,history,cd,pwd,exec,kill,export,read,echo

The implementation of these are in seperate files

Author
Michail-Panagiotis Bofos
Parameters
cmdthe command we want to check if it is a built in
Returns
zero if we want to quit, a positive if its a built in and minus one if its not a built in command
